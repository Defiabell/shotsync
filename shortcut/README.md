# iOS 快捷指令：分享图片到 shotsync

iOS Safari 不支持 PWA 作为系统分享目标，用快捷指令补齐「任意 App 分享 → 进池」。

## 步骤
1. 快捷指令 App → 新建 → 重命名「传到 shotsync」
2. 设置（ⓘ）→ 打开「在分享表单中显示」→ 接受类型只勾「图像」
3. 加动作「转换图像」→ 格式 JPEG（**必做**，把 HEIC 归一化，避免相册裂图）
4. 加动作「获取 URL 的内容」：
   - URL：`https://<你的域名>/api/upload`
   - 方法：POST
   - 请求头：`Authorization` = `Bearer <你的 AUTH_TOKEN>`
   - 请求体：表单（multipart），加一个字段：
     - 键 `full`，类型「文件」，值 = 上一步「转换后的图像」
5. （可选）加动作「显示通知」展示返回的 id 确认成功

## 用法
任意 App 看到图 → 分享 → 选「传到 shotsync」→ 数秒后相册可见。
多选图片时快捷指令会对每张依次执行。

## 说明
此路径不单独生成缩略图（`hasThumb=false`），相册会回退拉原图显示，手动分享量小可接受。
